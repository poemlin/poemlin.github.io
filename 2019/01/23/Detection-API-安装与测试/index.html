Object Detection API 安装

下载Tensorflow/models

Tensorflow在github上的主页是https://github.com/tensorflow，该主页上有一个有名的项目tensorflow/models，该项目存放了很多有趣的官方模型，目标检测官方模型就是在这里面。

下载后得到的文件内有很多文件夹，用于目标检测的只需要reserch文件夹下的object detection和slim文件夹，object detection存放了用于目标检测的工具和脚本，slim则是著名的图像分类模型，里面存放了很多已有的网络模型。最终项目结构为：

protoc编译

object_detection/protos/中存放了很多.proto文件，需要用protoc对它们进行编译成python文件。需要注意的是，系统protoc的版本必须大于2.6，否则编译会出现错误。

若版本低于2.6，需去protobuf官网https://github.com/google/protobuf/releases 下载已经编译好的文件包 。
得到.zip压缩包，解压后得到两个文件夹，将bin文件夹里的protoc文件覆盖复制到系统原来的protoc位置

这是由于终端上的 protoc 版本 与python库内的protobuf版本不一样。我的终端版本是3.6(最新)，但是python的protobuf版本是3.5，所以我们只需要更新python的protobuf到最新即可：
