文件的绝对路径与相对路径

绝对路径

文件的绝对路径当然是指：文件在硬盘上真正存在的路径。D:\Hexo\source_posts\a.md 就是一个绝对路径，告诉我们a.md这个文件在这个位置，我们不需要知道其他任何信息就可以根据绝对路径判断出文件的位置。但要注意的是这个硬盘是我们自己的硬盘。当其他人使用时就会出错。
